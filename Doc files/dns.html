DNS- What is the Internet

Domain Name System

Associates web names with the corresponding addresses. Devices uses DNS to lookup domain names and get the associated IP addresses. DNS servers are distributed by zones, by hierarchy and by .net, .org, .gov etc

DNS are designed to be on open public communication protocol and therefore are susceptible to hacks, for example, DNS spoofing in which IP address of a reliable/trusted website is assigned to phishing website.
