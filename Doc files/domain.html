Domain Names & ICANN- What is the Internet

Domain Names

Domain name is the address where a website is hosted. This address is also called IP address, which is a 32 bit, string of number, since it’s hard to remember numbers, we use domain name as webpage address. These names can be alpha numeric and each name/address is unique. Domain name, for example banana.com is made up of two parts, a second level domain (SLD) and a top level domain (TLD).  A second level domain is the name of the website, “banana” in the above example. Everything that comes after the dot is a top level domain, sometimes called “domain extension”. Together, SLD and TLD form what is considered a domain name.

ICANN

A non-profit organization that is responsible for Internet Protocol (IP) address space allocation, protocol identifier assignment, generic (gTLD) and country code (ccTLD) Top-Level Domain name system management, and root server system management functions.CANN is responsible for coordinating the management of the technical elements of the DNS to ensure universal resolvability so that all users of the Internet can find all valid addresses. It does this by overseeing the distribution of unique technical identifiers used in the Internet's operations, and delegation of Top-Level Domain names (such as .com, .info, etc.)
