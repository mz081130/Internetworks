W3C and HTML/CSS- What is the Internet

W3C

Founded in 1944, The World Wide Web Consortium (W3C) is an organization that works with other agencies to develop web standards. W3C's primary activity is to develop protocols and guidelines that ensure long-term growth for the Web. W3C's standards define key parts of what makes the World Wide Web work.

W3 standards allow developers to the potential to build rich interactive experiences, powered by vast data stores, that are available on any device. W3C and its partners created new technologies including CSS, SVG, WOFF, the Semantic Web stack, XML, and a variety of APIs.

HTML & CSS

Hyper Text Markup Language

A language that shows the data/images/videos should look like on a webpage when rendered on a computer or on a mobile device.

HTML consists of a series of elements, which you use to enclose, or wrap, different parts of the content to make it appear a certain way, or act a certain way. The enclosing tags can make a word or image hyperlink to somewhere else, can italicize words, can make the font bigger or smaller, and so on. For example, take the following line of content:

I am an Alien, watch out!

If we wanted the line to stand by itself, we could specify that it is a paragraph by enclosing it in paragraph tags:

<p>I am an Alien, watch out!</p>

Cascading Style Sheets is used for making the webpage more personalized, interactive and user friendly. The Text of the page is included directly in the code, while images and videos have separate path/URL and need to be requested and display as they being delivered.
